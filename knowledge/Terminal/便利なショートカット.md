# 主要なショートカット
新規タグの作成：command + t

タブの移動：command + → or ←

タブを閉じる：command + w

タブの左右分割：commnad + d

タブの上下分割：commnad + shift + d

分割したタブの移動：commnad + [ or ]

検索：command + f

検索にHitした文字をたどる：Command + G（Command + Shift + G）

検索にHitした単語から選択範囲を広げる：tab（Shift + Tab）
